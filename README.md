# Subversion: The Traitors

A web-based game platform inspired by The Traitors, featuring real-time video chat, voting mechanics, and multiple game phases.

## Features
- Real-time video/audio using Jitsi Meet
- Multiple room system for private conversations
- Host controls with production mode
- Voting and elimination mechanics
- Ghost player system
- Viewer mode with room switching

## Setup
1. Configure Firebase in `firebase-config.js`
2. Deploy to Firebase Hosting
3. Access via the deployed URL

## Roles
- **Host**: Full game control (Main Host or Production mode)
- **Player**: Participate in the game
- **Viewer**: Watch-only access
- **Ghost**: Former players who can still watch
